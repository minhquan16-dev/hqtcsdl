# Dashboard phân tích thị trường tuyển dụng IT

Frontend React Vite cho dashboard phân tích thị trường tuyển dụng IT Việt Nam.
Ứng dụng dùng JavaScript, shadcn/ui, Tailwind CSS, axios, TanStack React Query, React Router và Recharts.

## Chạy local

Tạo `.env` từ `.env.example` nếu cần đổi URL backend:

```env
VITE_API_BASE_URL=http://localhost:3001
```

Cài dependencies và chạy dev server:

```bash
npm install
npm run dev
```

## Kiểm tra

```bash
npm run lint
npm run build
```
